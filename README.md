# mid-qr dist branch

Built artefacts — do not edit manually. Source is on main.

## npm install from GitHub dist branch
```bash
npm install github:Mid-D-Man/mid-qr#dist
```

## Blazor
Copy `blazor/wasm/` → `wwwroot/wasm/`
Copy `blazor/js/`   → `wwwroot/js/`

The `site/` directory contains a pre-assembled version of the
website matching the gh-pages deployment.
